# Credit Risk EDA | Análise Exploratória de Risco de Crédito

Projeto de análise exploratória de dados (EDA) focado na investigação de padrões relacionados ao risco de crédito, utilizando técnicas de análise de dados para compreender fatores associados à inadimplência e comportamento financeiro.

O objetivo é transformar dados brutos em insights que auxiliem na avaliação de risco e na tomada de decisão em contextos financeiros.

---

## Visão Geral do Projeto

A análise busca responder perguntas relevantes para o contexto de crédito, como:

- Quais características estão associadas ao risco de inadimplência?  
- Existem padrões demográficos ou financeiros relevantes?  
- Como variáveis econômicas e comportamentais influenciam o risco?  
- Quais perfis apresentam maior probabilidade de default?  

O projeto simula um cenário real de análise em instituições financeiras ou empresas que concedem crédito.

---

## Objetivo da Análise

- Explorar dados relacionados a concessão de crédito  
- Identificar padrões associados ao risco financeiro  
- Aplicar técnicas de análise exploratória de dados (EDA)  
- Gerar insights que possam apoiar decisões de crédito  
- Desenvolver raciocínio analítico aplicado ao contexto financeiro  

---

## Abordagem Analítica

A análise foi conduzida seguindo etapas estruturadas:

1. Compreensão do contexto do problema  
2. Exploração inicial da base de dados  
3. Limpeza e tratamento das variáveis  
4. Análise exploratória e identificação de padrões  
5. Construção de visualizações  
6. Interpretação dos resultados  
7. Síntese de insights relevantes  

---

## Técnicas Aplicadas

- Análise exploratória de dados (EDA)  
- Estatística descritiva  
- Identificação de outliers  
- Análise de distribuição de variáveis  
- Visualização de dados  
- Investigação de relações entre variáveis  

---

## Possíveis Aplicações de Negócio
